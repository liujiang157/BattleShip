### 操作方法：

- **操作：**
- KeyBoard_I:无敌
  
- KeyBoard_A:超级火力
  
- KeyBoard_M:打开/关闭音乐
  
- KeyBoard_L：复活
  
- KeyBoard_C：清空敌军
  
- KeyBoard_F：生成敌军
  
- KeyBoard_Ctrl：开火
  
- **移动--支持八方行走**：
- KeyBoard_Up：上
  
- KeyBoard_Down：下
  
- KeyBoard_Left：左
  
- KeyBoard_Right：右
